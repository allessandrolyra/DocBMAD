# BMAD Framework - Documentação Oficial

Este repositório contém a documentação interativa e visual do **BMAD Framework** (B·M·A·D Method).

O BMAD é um framework de agentes de IA especializado em transformar assistentes genéricos em equipes completas de produto, design, desenvolvimento e QA.

## 🚀 Como Visualizar

1. Faça o clone do repositório:
   ```bash
   git clone [url-do-seu-repositorio]
   ```
2. Abra o arquivo `index.html` em qualquer navegador moderno.

## 📦 Estrutura do Projeto

- `index.html`: Página principal com design responsivo, sidebar interativa e arquitetura SVG.
- `.gitignore`: Configurações para manter o repositório limpo.
